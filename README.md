# HCM2024 — Hypertrophic Cardiomyopathy Clinical Guide

A clinician-facing web application encoding the **2024 AHA/ACC/AMSSM/HRS/PACES/SCMR Guideline for the Management of Hypertrophic Cardiomyopathy**.

> Ommen SR, Ho CY, Asif IM, et al. *Circulation*. 2024;149:e1239–e1311.

## Features

- **Diagnosis** — Clinical evaluation, echocardiography, CMR, cardiac CT, heart rhythm assessment, exercise stress testing, genetics & family screening, genotype-positive/phenotype-negative management
- **SCD Risk Assessment** — Adult and pediatric risk stratification, ICD patient selection algorithm, device selection considerations
- **Management** — Pharmacological and invasive treatment of obstructive HCM, nonobstructive HCM, advanced heart failure, atrial fibrillation, ventricular arrhythmias
- **Lifestyle** — Exercise and sports participation, occupational considerations, pregnancy management, comorbidity management
- **Ask NotebookLM** — AI-powered Q&A with brief and explanatory response modes

## Tech Stack

Next.js 14+ (App Router) · TypeScript · Tailwind CSS · Zustand · react-markdown

## Getting Started

```bash
npm install
npm run dev
```

## Disclaimer

This tool is for educational and clinical reference purposes only. It does not replace clinical judgment. Always verify recommendations against the full guideline and local protocols.
